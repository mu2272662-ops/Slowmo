# 🎬 SlowMo Pro

A professional Video Slow Motion Maker with a beautiful GUI interface.

## Features

- ✨ Simple and intuitive GUI
- 🎥 Support for multiple video formats (MP4, AVI, MOV, MKV, etc.)
- ⚡ Adjustable slow motion speed (2x to 10x)
- 📊 Real-time progress tracking
- 📝 Detailed logging
- 🚀 Built-in FFmpeg (no external installation needed)

## Installation

### For Users (EXE)
1. Download the latest `SlowMoPro.exe` from Releases
2. Double-click to run (no installation required)

### For Developers
```bash
git clone https://github.com/yourusername/slowmo-pro
cd slowmo-pro
pip install -r requirements.txt
python slowmo_app.py
